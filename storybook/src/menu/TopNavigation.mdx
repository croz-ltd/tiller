# Top Navigation

`Top Navigation` gives you a complete styled navigation bar on the top of your website, ready to store links to your pages, custom actions, dropdown and even subitems.

The component comes in 3 possible variants:
 - primary color styled (defined in your config file)
 - dark version
 - light version

If you aren't a fan of a full styled navigation and want only the navigation itself to be styled according to your primary color, you can edit your dropdowns to be
themed differently via the props sent to the dropdown menu.

**Limitations:**

Due to Storybook's limitations, the shown code will not be functional unless you replace the following:
  - `TopNavigationNavigation` -> `TopNavigation.Navigation`

The navigation items are wrapped in a `TopNavigation.Navigation` container.
