# Sidebar Navigation

`Sidebar Navigation` gives you a complete styled side navigation for your website, ready to store links to your pages, subitems, dropdown and even bottom actions for most frequently
visited links.

The component comes in 3 possible variants:
 - primary color styled (defined in your config file)
 - dark version
 - light version

If you aren't a fan of a full styled navigation and want only the navigation itself to be styled according to your primary color, you can edit your dropdowns to be
themed differently via the props sent to the dropdown menu.

The best possible documentation for this component would be the [Props documentation](#sidebar-navigation-props)
and the code of the stories listed below.

**Limitations:**
 - Due to Storybook's limitations, the shown code will not be functional unless you replace the following:
  - `SidebarNavigationItem` -> `SidebarNavigation.Item`
  - `SidebarNavigationDropdown` -> `SidebarNavigation.Dropdown`
  - `SidebarNavigationDropdownItem` -> `SidebarNavigation.Dropdown.Item`
  - `SidebarNavigationBottomAction` -> `SidebarNavigation.BottomAction`
  - `SidebarNavigationSubItem` -> `SidebarNavigation.SubItem`
