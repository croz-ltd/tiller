# Password Input Field

The `PasswordInputField` component is an input field modified to support password inputs.

It is most commonly used on login forms (see [Login](/story/patterns-login--simple) pattern).

By editing the `help` prop you can modify the message which pops up when caps lock is turned on.

All props are described in detail on the [Password Input Field Props](#password-input-field-props) section of the documentation.
