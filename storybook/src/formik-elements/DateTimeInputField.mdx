# Date Time Input Field

`DateTimeInputField` is an input field for selecting both a date and a time.

Allows choosing date then time. There are 4 steps available (year, date, hour and minute), so tabs are required to visually distinguish date/time steps.

## **Prerequisites for functioning**:
 - Intl Provider (details [here](/docs/component-library-intl-intl--basic))

`DateTimeInputField` component has one unique prop:
 - `name` (_string_): the accessor value for the field component (for validation, fetching, etc.).
