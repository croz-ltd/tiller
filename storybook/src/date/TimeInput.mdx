# Time Input

Time Input allow users to enter a specific time value. They can be used for a wide range of scenarios.

Common use cases include:

- Setting an alarm
- Scheduling a meeting

The component accepts the following types for **value** prop:

- null
- string type - e.g. _"21:30"_, _"2020-11-20T11:21:28.635778"_, _"2020-11-20T11:21:28.63602+05:00"_

For integration with Formik see docs [here](/docs/component-library-formik-elements-timeinputfield--with-label#timeinputfield).

Props are described on the [Time Input Props section](/docs/component-library-date-timeinput--disabled#time-input-props)
of the documentation.
