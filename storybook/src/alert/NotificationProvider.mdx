# Notification Provider

`NotificationProvider` component is used to summon a notification on the desired page by defining
the look of the `Notification` component.

## Usage: Step-by-step Instructions

  **1.** Wrap your app (or the area where the rendering of the components is enabled) in the `NotificationProvider`
component <br /> <br />
  **2.** On the page which displays the notification (either by executing a task or by any desired function) define
a `useNotificationContext()` function imported from our library and assign it to a constant. <br /> <br />
  **3.** By using a **`.push`** method, define the notification you would like to display by using props
  from the `Notification` component (**props are defined [here](/docs/component-library-alert-notification--with-actions-right#notification-props)**) <br /> <br />

#### For Example:

```tsx
const notifications = useNotificationContext()

notifications.push({
  **title: "Example Notification",
  description: "Example notification description."
  icon: { type: "check-circle", variant: "outline", color: "green" },
  onDismiss:
});
```

All props are described on the [Notification Provider Props section](/docs/component-library-alert-notificationprovider--example#notification-provider-props)
of the documentation.

The `<Content/>` part of the code is the following function:

```tsx
function Content() {
    const notifications = useNotificationContext();

    const simple: NotificationProps = {
    title: "Simple",
    };

    const withSubtitle: NotificationProps = {
    title: "Simple",
    content: "Subtitle",
    };

    const withIcon: NotificationProps = {
    title: "Simple",
    content: "Subtitle",
    icon: <Icon type="check-circle" size={6} variant="fill" className="text-emerald-500" />,
    };

    const withActionsBelow: NotificationProps = {
    title: "With Actions Below",
    content: "Anyone with a link can now view this file.",
    icon: <Icon type="check-circle" size={6} variant="fill" className="text-emerald-500" />,
    actions: [<Link key={0}>Undo</Link>, <Link key={1}>Dismiss</Link>],
    onDismiss: () => {},
    };

    const withButtonsBelow: NotificationProps = {
    title: "With Buttons Below",
    content: "Anyone with a link can now view this file.",
    icon: <Icon type="check-circle" size={6} variant="fill" className="text-emerald-500" />,
    actions: [
    <Button key={1} variant="filled">
    Accept
    </Button>,
    <Button key={0} variant="text">
    Dismiss
    </Button>,
    ],
    onDismiss: () => {},
    };

    const withActionsRight: NotificationProps = {
    title: "With Actions Right",
    content: "Anyone with a link can now view this file.",
    icon: <Icon type="check-circle" size={6} variant="fill" className="text-emerald-500" />,
    closeButton: false,
    actions: [<Link key={0}>Reply</Link>],
    onDismiss: () => {},
    };

    return (
        <div className="flex flex-col w-1/4 space-y-2">
            <Button onClick={() => notifications.push(simple)}>Simple</Button>
            <Button onClick={() => notifications.push(withSubtitle)}>With Subtitle</Button>
            <Button onClick={() => notifications.push(withIcon)}>With Icon</Button>
            <Button onClick={() => notifications.push(withActionsBelow)}>With Actions Below</Button>
            <Button onClick={() => notifications.push(withButtonsBelow)}>With Buttons Below</Button>
            <Button onClick={() => notifications.push(withActionsRight)}>With Actions Right</Button>
        </div>
    );
}
```
