# CheckboxGroup

`Checkbox Group` component represents a checkbox group that enables selection of single or multiple options.

If the `required` attribute is set to true, at least one checkbox must be selected.

(for integration with Formik see docs [here](/docs/component-library-formik-elements-checkboxgroupfield--simple))
